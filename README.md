# 🎮 Kreuz-und-Kreis (Tic-Tac-Toe) in Python

Ein konsolenbasiertes Kreuz-und-Kreis-Spiel, entwickelt mit Python.\
Das Projekt dient als Lernprojekt zur Vertiefung von Spiel-Logik,
Funktionen und Eingabevalidierung.

------------------------------------------------------------------------

## 🚀 Funktionen

-   Zwei-Spieler-Modus (X und O)
-   Automatische Gewinnprüfung
-   Erkennung eines Unentschiedens
-   Schutz vor ungültigen Eingaben
-   Saubere Funktionsstruktur

------------------------------------------------------------------------

## 🛠 Technologien

-   Python 3
-   Keine externen Bibliotheken notwendig

------------------------------------------------------------------------

## ▶️ Installation & Start

1.  Repository klonen:

``` bash
git clone https://github.com/DEIN-BENUTZERNAME/tic-tac-toe-python.git
```

2.  In den Projektordner wechseln:

``` bash
cd tic-tac-toe-python
```

3.  Spiel starten:

``` bash
python main.py
```

------------------------------------------------------------------------

## 🎯 Spielanleitung

-   Spieler X beginnt.
-   Gib eine Zeile und eine Spalte (0--2) ein.
-   Ziel ist es, drei eigene Zeichen in einer Reihe, Spalte oder
    Diagonale zu platzieren.

Beispiel:

Spieler X, wähle eine Zeile (0-2): 1\
Spieler X, wähle eine Spalte (0-2): 2
